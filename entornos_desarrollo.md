# Entornos de desarrollo

Los entornos de desarrollo son herramientas que ayudan al programador durante el proceso de creación, prueba, mantenimiento y mejora del software.

En un proyecto informático no solo es importante escribir código. También es necesario organizar los archivos, controlar los cambios, probar el funcionamiento del programa y documentar las decisiones tomadas.

## IDE
un IDE es un software de desarrollo para programacion en el qu puedes editar, compilar o depurar codigo

## Control de versiones
Un sistema de control de versiones es una herramienta para guadar versiones de codigos o de archivos. Es útil porque te permite revertir cambos, tener un historial y control a largo plazo de lo que has estado haciendo.


## Pruebas de software
Las pruebas de software son procesos de evaluación que verifican que una aplicación funciona correctamente.
Las pruebas unitarias son métodos automatizados que verifican que la parte más pequeña de código testeable.
Las pruebas automatizadas permiten una ejecución rápida  de casos de prueba, identificando errores de forma temprana.

## Repositorios remotos
Un repositorio remoto es una versión del proyecto alojada en GitHub. Sirve para el trabajo colaborativo copias de seguridad, etc.
